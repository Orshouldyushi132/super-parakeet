# MIDI小節切り替え動画ツール

このツールは、MIDIファイルに含まれるノーツを1小節ごとに固定表示し、発音中のノーツだけ色や光り方、アニメーションを変えながら、音声なしのMP4動画として曲全体を書き出すアプリです。

Windows ではデスクトップ版、Mac とスマホではブラウザ版を使えます。どちらも同じ描画エンジンを使うので、見た目はほぼ同じです。

## できること

- 1小節ごとの固定表示をつないだ動画を書き出し
- 発音中ノーツだけ色変更
- 背景色、通常ノーツ色、発音中ノーツ色、発光色、アニメ色、輪郭色の変更
- 角の形を `四角 / 丸角 / カプセル` から選択
- 光り方を `なし / ソフト / ぼんやり / ブルーム / ネオン / オーラ / 輪郭 / シャドウ / プリズム` から選択
- アニメーションを `なし / 点滅 / ポップ / 走査線 / ガタガタ / アーケード / パルス / ブレス / シマー / バウンス / エクスパンド / フリッカー / ウェーブ` から選択
- 残像を `おまかせ / なし / 枠だけ / 塗りだけ / 枠+塗り` から選択
- 発光、アニメ、速度、残像の強さをスライダーで調整
- テーマプリセット
  `モノクロフラッシュ / アイスブルー / サンセット / エメラルド / マゼンタネオン / ゴールドスパーク / アーケード`

## 表示仕様

- 出力解像度: `1920x1080`
- 背景色の初期値: 黒
- 通常ノーツ色の初期値: 黒よりの灰色
- 発音中ノーツ色の初期値: 白
- 表示方式: 1小節ごとの固定表示を曲全体で順番につなぐ
- 音声: なし
- 複数トラック: まとめて1つの表示に統合

## セットアップ

```powershell
python -m pip install -r requirements.txt
```

## Windowsデスクトップ版の起動

```powershell
python main.py
```

または `run.bat` をダブルクリックしてください。

## Mac / スマホ対応のブラウザ版

ブラウザ版を使うと、Mac でもスマホでも同じ画面を開けます。

### 起動

```powershell
python web_main.py --host 0.0.0.0 --port 8000
```

Windows では `run_web.bat` を実行しても起動できます。

### 開き方

- 同じPCで使う: `http://127.0.0.1:8000`
- Macで使う: ブラウザで上のアドレスを開く
- スマホで使う: PCとスマホを同じWi-Fiに接続し、スマホのブラウザで `http://PCのIPアドレス:8000` を開く

例:

```text
http://192.168.1.12:8000
```

ブラウザ版では、MIDIアップロード、プレビュー、テーマ変更、色変更、MP4書き出しまで行えます。

## EXEの作成

```powershell
python -m pip install -r requirements-build.txt
python -m PyInstaller --noconfirm MidiMeasureVideoExporter.spec
```

または `build_exe.bat` を実行してください。

生成される実行ファイル:

```text
dist\MidiMeasureVideoExporter\MidiMeasureVideoExporter.exe
```

配布するときは、`_internal` を含む `dist\MidiMeasureVideoExporter` フォルダごとまとめて扱ってください。

## 使い方

1. MIDIファイルを開きます。
2. 再生ボタンで小節ごとの切り替え表示をプレビューします。
3. 右側の見た目設定で、テーマ、角の形、色、光り方、アニメーション、残像を調整します。
4. `MP4を書き出す` を押すと、現在の設定のまま動画を書き出します。

## 補足

- 拍子変更があるMIDIにも対応しており、その拍子に合わせて小節ごとに表示が切り替わります。
- テーマを選んだあとに色やスタイルを個別変更すると、自動的に `カスタム` 扱いになります。
- Windows の EXE 版には MP4 書き出しに必要な FFmpeg ランタイムが含まれています。
- ブラウザ版は見た目確認用プレビューを軽く返し、最終書き出しは `1920x1080` で行います。
