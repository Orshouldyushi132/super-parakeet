# MIDIを動画にするやつ

MIDI ファイルのノーツを映像として表示し、動画として書き出すためのアプリです。  
Windows 向けのデスクトップ版と、Mac / Windows / スマホから使えるブラウザ版があります。

## 表示モード

- `演奏ビュー`
  再生バーの近くにあるノーツだけを見せるモードです。時間、拍、小節、統計、コードを重ねて表示できます。
- `固定ページ`
  1ページに小節を固定表示して、ページを切り替えながら見せるモードです。

## 主な機能

- MIDI の読み込みとプレビュー
- 動画の書き出し
- 既定値 `120FPS` の高フレームレート出力
- `H.264 MP4 / MOV / 連番PNG` の書き出し形式切り替え
- `横動画 / 縦動画` の切り替え
- `4K / 1440p / 1080p / 720p` の解像度切り替え
- 発音中ノーツの色変更
- 発光、残像、枠アニメーション、フェード演出
- ノーツサイズ、枠の太さ、余白、残像時間などの細かい調整
- コード表示とノーツ表示の間にある歌詞スペースの高さ調整
- `演奏ビュー` での再生バー表示
- `演奏ビュー` での時間 / 拍 / 小節表示
- `演奏ビュー` での統計表示
- `演奏ビュー` でのコード表示
- 未再生ノーツを隠す表示
- 表示小節数の変更
- 文字色の変更
- 背景透過の書き出し
- 見た目設定のプリセット保存 / 読み込み / 削除
- 保存したプリセットのデスクトップ版 / ブラウザ版共有
- MIDI データからの自動音声生成
- 追加の音声ファイルを重ねた再生 / 書き出し
- MIDIファイルや音声ファイルのパス表示方式の切り替え

## 演奏ビューでできること

- 表示方式を `演奏ビュー / 固定ページ` から切り替え
- 表示する小節数を `1〜8小節` で変更
- 歌詞スペースの高さを自由に変更
- 未再生ノーツを隠す
- 再生バーの表示 / 非表示
- 時間と拍の表示 / 非表示
- 小節ガイドの表示 / 非表示
- 統計表示の表示 / 非表示
- コード表示の表示 / 非表示
- 背景透過の切り替え

- 発音中ノーツ色

- アニメ色

- 文字色

## プリセット保存

- 現在の見た目設定を名前付きプリセットとして保存できます
- 保存したプリセットはデスクトップ版とブラウザ版の両方で使えます
- ユーザープリセットは次の場所に保存されます

```text
%USERPROFILE%\.midi_measure_video_exporter\user_presets.json
```

## 動作環境

- Python 3.12 推奨
- Windows デスクトップ版: `tkinter`
- ブラウザ版: Mac / Windows / スマホのブラウザ
- 動画書き出し: `imageio-ffmpeg`
- 描画品質向上: `aggdraw`

## インストール

```powershell
python -m pip install -r requirements.txt
```

## Windows デスクトップ版の起動

```powershell
python main.py
```

または `run.bat` を実行してください。

## ブラウザ版の起動

```powershell
python web_main.py --host 0.0.0.0 --port 8000
```

Windows では `run_web.bat` でも起動できます。

### 開き方

- 同じ PC で使う場合: `http://127.0.0.1:8000`
- Mac やスマホで使う場合:
  同じ Wi-Fi に接続した PC でブラウザ版を起動し、PC の IP アドレスを開いてください

例:

```text
http://192.168.1.12:8000
```

## 使い方

1. MIDI ファイルを読み込みます
2. 表示方式を選びます
3. 必要なら表示小節数やオーバーレイを調整します
4. 見た目設定を調整します
5. 必要なら歌詞スペースや音声設定を調整します
6. 必要ならプリセット名を入力して保存します
7. 動画を書き出します

## FPS について

- 初期値は `120FPS` です
- 出力時は `1〜240FPS` の範囲で変更できます
- 滑らかさ重視の設定でプレビューと書き出しを行います

## 書き出し設定について

- `書き出し` タブから出力形式、動画の向き、解像度、FPS をまとめて変更できます
- デフォルトは `4K / 横動画 / 120FPS` です
- プレビュー比率も書き出し設定に合わせて自動で切り替わります
- デスクトップ版では `MIDI音を鳴らす / 追加音声ファイル / ループ / 音量` もここで設定できます
- `基本` タブの `パス表示` から、ファイル名だけ、フォルダ名 + ファイル名、ディレクトリパス、フルパス、非表示を選べます

## 背景透過について

- 背景透過を ON にすると、透過背景で描画されます
- 背景透過を書き出せるのは `MOV` と `連番PNG` です
- `H.264 MP4` では背景透過は自動で OFF になります

## EXE の作成

```powershell
python -m pip install -r requirements-build.txt
python -m PyInstaller --noconfirm MidiMeasureVideoExporter.spec
```

または `build_exe.bat` を実行してください。

生成される実行ファイル:

```text
dist\MidiMeasureVideoExporter\MidiMeasureVideoExporter.exe
```

配布時は `_internal` を含む `dist\MidiMeasureVideoExporter` フォルダごと一緒にしてください。

## 補足

- 複数トラックの MIDI はまとめて表示します
- デスクトップ版では再生時に MIDI 音を自動で鳴らせます
- `H.264 MP4` と `MOV` は音声つきで書き出せます
- `連番PNG` は画像に加えて同じフォルダへ `WAV` を出力します
- 出力解像度は `4K / 1440p / 1080p / 720p` から選べます
- ブラウザ版の表示速度は端末性能や通信状況の影響を受けます
