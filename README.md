# MIDIを動画にするやつ

MIDIファイル内のノーツを、小節ごとの固定レイアウトで表示し、発音中のノーツだけ色や光り方やアニメーションを変えて、そのまま音声なしのMP4動画として書き出すアプリです。

Windows向けのデスクトップ版と、Mac / スマホ / Windows で使えるブラウザ版の両方を用意しています。

## できること

- MIDIを読み込んで、小節ごとの固定表示をプレビュー
- 発音中ノーツの色変更、発光、枠アニメーション、残像、フェードアウトを調整
- ノーツの長さ、高さ、余白、枠の太さ、残像の残る時間などを細かく変更
- 角の形を `四角 / 丸角 / カプセル` から選択
- 見た目設定をプリセットとして保存・再利用・削除
- 保存したプリセットをデスクトップ版とブラウザ版の両方で共有
- 既定値 `120FPS` で、より滑らかなプレビュー確認と動画書き出し
- 音声なしの `1920x1080 MP4` を書き出し

## 主な見た目設定

- 背景色
- 通常ノーツ色
- 発音中ノーツ色
- 光色
- アニメーション色
- 枠色
- 光り方
- アニメーション種類
- 残像の種類
- ノーツの長さ
- ノーツの高さ
- 横余白 / 縦余白
- 通常枠 / 発音中枠 / 残像枠の太さ
- 残像の持続時間
- 残像フレームの広がり
- ノーツ終了時のフェード方法とフェード時間

## プリセット保存

- デスクトップ版もブラウザ版も、現在の見た目設定を名前付きプリセットとして保存できます
- 保存済みプリセットは両方のUIに自動で表示されます
- ユーザープリセットは次の場所に保存されます

```text
%USERPROFILE%\.midi_measure_video_exporter\user_presets.json
```

## 動作環境

- Python 3.12 推奨
- Windowsデスクトップ版: `tkinter` を使用
- ブラウザ版: Mac / Windows / スマホのブラウザで利用可能
- 動画書き出しには `imageio-ffmpeg` を使用
- 描画品質向上のため `aggdraw` を使用

## インストール

```powershell
python -m pip install -r requirements.txt
```

## Windowsデスクトップ版の起動

```powershell
python main.py
```

または `run.bat` を実行してください。

## ブラウザ版の起動

```powershell
python web_main.py --host 0.0.0.0 --port 8000
```

Windowsでは `run_web.bat` でも起動できます。

### 開き方

- 同じPCで使う場合: `http://127.0.0.1:8000`
- Macやスマホで使う場合:
  同じWi-Fiに接続したPCでブラウザ版を起動し、PCのIPアドレスを開いてください

例:

```text
http://192.168.1.12:8000
```

## 使い方

1. MIDIファイルを読み込みます
2. 再生しながら小節ごとの表示を確認します
3. 見た目設定を調整します
4. 必要ならプリセット名を入力して保存します
5. `MP4を書き出し` で動画を保存します

## FPSについて

- 初期値は `120FPS` です
- ブラウザ版もデスクトップ版も、滑らかさ重視の設定になっています
- 必要に応じて `1〜240FPS` の範囲で変更できます

## EXEの作成

```powershell
python -m pip install -r requirements-build.txt
python -m PyInstaller --noconfirm MidiMeasureVideoExporter.spec
```

または `build_exe.bat` を実行してください。

生成される実行ファイル:

```text
dist\MidiMeasureVideoExporter\MidiMeasureVideoExporter.exe
```

配布時は `_internal` を含む `dist\MidiMeasureVideoExporter` フォルダごと一緒にしてください。

## 補足

- 1つのMIDI内に複数トラックがあっても、表示はまとめて扱います
- 書き出し動画は音声なしです
- ブラウザ版のプレビューはネットワークや端末性能に応じて表示速度が変わります
- 出力動画の解像度は現在 `1920x1080` 固定です
