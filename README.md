# MIDIを動画にするやつ

MIDIファイルのノーツを映像として見せるためのアプリです。  
Windows向けのデスクトップ版と、Mac / Windows / スマホから使えるブラウザ版があります。

今は次の2つの表示モードを使えます。

- `演奏ビュー`
  再生バーの近くにあるノーツだけを見せるモードです。未再生ノーツを隠したり、時間・拍・小節・コード・統計を重ねたりできます。参考画像のような見せ方を作りたいときはこちらが向いています。
- `固定ページ`
  1ページに小節を固定表示して、ページを切り替えながら見せるモードです。

## 主な機能

- MIDIの読み込みとプレビュー
- MP4動画の書き出し
- 既定値 `120FPS` での高フレームレート出力
- 発音中ノーツの色変更
- 発光、残像、枠アニメーション、フェード演出
- ノーツサイズ、枠の太さ、余白、残像時間などの細かい調整
- `演奏ビュー` での再生バー表示
- `演奏ビュー` での時間 / 拍 / 小節表示
- `演奏ビュー` でのコード名表示
- `演奏ビュー` での統計表示
- 未再生ノーツを隠す表示
- 表示する小節数の変更
- 見た目設定のプリセット保存 / 読み込み / 削除
- 保存したプリセットのデスクトップ版 / ブラウザ版共有

## 演奏ビューでできること

- 表示方式を `演奏ビュー / 固定ページ` から切り替え
- 表示する小節数を `1〜8小節` で変更
- 未再生ノーツを隠す
- 再生バーの表示 / 非表示
- 時間と拍の表示 / 非表示
- 小節ガイドの表示 / 非表示
- 統計表示の表示 / 非表示
- コード表示の表示 / 非表示

## 主な見た目設定

- 背景色
- 通常ノーツ色
- 発音中ノーツ色
- 光色
- アニメーション色
- 枠色
- 角の形
- 光り方
- アニメーション種類
- 残像の種類
- ノーツの長さ
- ノーツの高さ
- 横余白 / 縦余白
- 通常枠 / 発音中枠 / 残像枠の太さ
- 残像の持続時間
- 残像フレームの広がり
- ノーツ終了時のフェード方法とフェード時間

## プリセット保存

- 現在の見た目設定を名前付きプリセットとして保存できます
- 保存したプリセットはデスクトップ版とブラウザ版の両方で使えます
- ユーザープリセットは次の場所に保存されます

```text
%USERPROFILE%\.midi_measure_video_exporter\user_presets.json
```

## 動作環境

- Python 3.12 推奨
- Windowsデスクトップ版: `tkinter`
- ブラウザ版: Mac / Windows / スマホのブラウザ
- 動画書き出し: `imageio-ffmpeg`
- 描画品質向上: `aggdraw`

## インストール

```powershell
python -m pip install -r requirements.txt
```

## Windowsデスクトップ版の起動

```powershell
python main.py
```

または `run.bat` を実行してください。

## ブラウザ版の起動

```powershell
python web_main.py --host 0.0.0.0 --port 8000
```

Windowsでは `run_web.bat` でも起動できます。

### 開き方

- 同じPCで使う場合: `http://127.0.0.1:8000`
- Macやスマホで使う場合:
  同じWi-Fiに接続したPCでブラウザ版を起動し、PCのIPアドレスを開いてください

例:

```text
http://192.168.1.12:8000
```

## 使い方

1. MIDIファイルを読み込みます
2. 表示方式を選びます
3. 必要なら表示小節数やオーバーレイを調整します
4. 見た目設定を調整します
5. 必要ならプリセット名を入力して保存します
6. `MP4を書き出し` で動画を保存します

## FPSについて

- 初期値は `120FPS` です
- 出力時は `1〜240FPS` の範囲で変更できます
- 滑らかさ重視の設定でプレビューと書き出しを行います

## EXEの作成

```powershell
python -m pip install -r requirements-build.txt
python -m PyInstaller --noconfirm MidiMeasureVideoExporter.spec
```

または `build_exe.bat` を実行してください。

生成される実行ファイル:

```text
dist\MidiMeasureVideoExporter\MidiMeasureVideoExporter.exe
```

配布時は `_internal` を含む `dist\MidiMeasureVideoExporter` フォルダごと一緒にしてください。

## 補足

- 複数トラックのMIDIはまとめて表示します
- 動画は音声なしです
- 出力解像度は現在 `1920x1080` 固定です
- ブラウザ版の表示速度は端末性能や通信状況の影響を受けます
